🎨 Icon Generator

Click Download under each size to save the PNG. You'll need all three for the Play Store and PWA manifest.

After downloading:
1. Save the files to your icons/ folder as icon-192.png, icon-512.png, icon-1024.png
2. The manifest.json and Play Store submission will use the 512px version
3. iOS App Store requires 1024px

Need to change the design? Edit the drawIcon() function in this file.
